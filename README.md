# ひらがな あそび

2歳児が楽しみながら「ひらがな」の音と形を一致させるためのブラウザゲームです。

**遊ぶ**: https://nnnnishi.github.io/hiragana-learning-game/

## 遊び方

音声で読み上げられたひらがなを、表示された選択肢の中からタップして答えます。

### レベル

| レベル | 選択肢 | ヒント |
|--------|--------|--------|
| レベル 1 | 2つ | 正解の文字が大きく薄く表示される |
| レベル 2 | 3つ | 「？」ボタンで一瞬だけ正解を表示 |
| レベル 3 | 5つ | ヒントなし |

### 特徴

- **絶対に否定しない** — 不正解でも「ブブー」などの音は鳴らさず「もういちど やってみよう！」と優しく誘導します
- **褒めちぎる** — 正解時は紙吹雪エフェクト + 音声で褒め言葉を再生します
- **5問正解で「やったね！」画面** — 達成感を演出するマイルストーン画面が表示されます
- **出題範囲の設定** — 設定画面（⚙️）から、行ごとに出題するひらがなを選べます
- **大きなボタン** — 幼児の誤タップを防ぐ大きめのレイアウト（iPad mini想定）
- **学習きろく** — 正答率を記録して成長を可視化できます（📊ボタン）

### 学習きろく機能

スタート画面の📊ボタンから確認できます。

- **合計回答数と総正答率** — これまでの全回答を集計
- **直近7日間の推移** — 日別の正答率を棒グラフで表示
- **文字別の正答率** — 行ごとにグループ化し、◎（80%以上）/ ○（60%以上）/ △（60%未満）で色分け
- **データ保存** — JSON形式でダウンロード可能
- **リセット** — 確認後にすべての記録を削除

データはブラウザのlocalStorageに保存され、直近90日分が自動保持されます。

### 適応的出題（トンプソンサンプリング）

出題する文字の選択に、学習履歴を活用したトンプソンサンプリングを採用しています。

- **苦手な文字ほど多く出題** — 不正解が多い文字ほど出題確率が高くなります
- **未出題の文字も自然に探索** — まだ出題されていない文字は一様分布として扱われ、適度に出題されます
- **確率的な選択** — 毎回決定的に同じ文字を出すのではなく、Beta分布からのサンプリングにより適度なランダム性を保ちます

仕組み: 各文字について Beta(不正解数+1, 正解数+1) からサンプリングし、最もスコアが高い文字を出題します。不正解が多いほどαパラメータが大きくなり、サンプリング値が高くなりやすくなります。

## 技術構成

- `index.html` 1ファイルで完結（Vanilla JS / CSS）
- 音声: [VOICEVOX](https://voicevox.hiroshiba.jp/) で事前生成したwavファイル（`./audio/`）
  - wavファイルが無い場合は Web Speech API にフォールバック
- ホスティング: GitHub Pages

## 音声ファイルの生成（開発者向け）

音声ファイルの再生成にはVOICEVOXが必要です。

1. [VOICEVOX](https://voicevox.hiroshiba.jp/) をインストール・起動
2. スクリプトを実行:

```bash
bash scripts/generate_audio.sh
```

ひらがな46文字 + 褒め言葉の音声が `./audio/` に生成されます。
