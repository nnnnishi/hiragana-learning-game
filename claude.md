## 概要
2歳児が楽しみながら「ひらがな」の音と形を一致させるための、非常にシンプルなブラウザゲーム。
ブラウザはipad miniの画面で実行することを想定

## ターゲットユーザー
- 2歳前後の幼児（文字を覚え始めの段階）
- 操作をサポートする保護者

## 開発方針 (Core Principles)
1. **絶対に否定しない**: 不正解でも「ブブー」などの音は鳴らさず、優しく正解へ誘導する。
2. **成功体験の重視**: 正解時は派手なエフェクトと音声で褒めちぎる。
3. **誤操作防止**: ボタンは大きく、余白を広く取り、誤タップを防ぐレイアウト。
4. **シンプルさ**: 1ファイル（index.html）で完結するVanilla JS構成。

## レベル設計
- **Level 1**: 選択肢2つ。正解の文字が大きくヒント表示されている。
- **Level 2**: 選択肢3つ。ヒントは隠し（「？」ボタンで表示）。
- **Level 3**: 選択肢5つ。ヒントなし。ランダム出題。

## 技術スタック
- HTML5 / CSS3 / JavaScript (Vanilla JS)
- 音声: VOICEVOX（事前生成wav） + Web Speech APIフォールバック
  - VOICEVOXスピーカー: ずんだもん (speaker=3)
  - wavファイルは `./audio/` に格納
  - `scripts/generate_audio.sh` で一括生成
- アイコン/フォント: 視認性の高いゴシック体（または教科書体）
- デプロイ先: GitHub Pages
